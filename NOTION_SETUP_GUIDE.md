# Notion 데이터베이스 ID 올바르게 추출하는 방법

## 문제 상황
"Provided ID is a page, not a database" 오류가 발생하는 경우, 데이터베이스가 아닌 페이지의 ID를 입력한 것입니다.

## 해결 방법

### 방법 1: 데이터베이스 URL에서 직접 추출

1. **데이터베이스 뷰로 전환**
   - Notion에서 데이터베이스를 열 때, 테이블 뷰나 보드 뷰로 표시되어야 합니다
   - 페이지 형태가 아니라 데이터베이스 형태여야 합니다

2. **URL 확인**
   - 데이터베이스 페이지의 주소창 URL을 확인하세요
   - 올바른 형식: `https://www.notion.so/workspace/데이터베이스ID?v=...`
   - 또는: `https://www.notion.so/데이터베이스ID`

3. **ID 추출**
   - URL에서 `?` 앞의 긴 문자열이 데이터베이스 ID입니다
   - 하이픈이 있으면 그대로 사용하세요
   - 예: `2c2379dc-8862-80c2-a6b4-da788c7b498e`

### 방법 2: 데이터베이스 속성에서 확인

1. 데이터베이스 페이지 열기
2. 우측 상단 `...` 메뉴 클릭
3. "Copy link" 또는 "링크 복사" 선택
4. 복사된 URL에서 ID 추출

### 방법 3: API로 데이터베이스 목록 조회 (고급)

데이터베이스 ID를 찾기 어려운 경우, Integration이 접근 가능한 모든 데이터베이스를 조회할 수 있습니다.

## 확인 사항

- [ ] 데이터베이스가 테이블/보드/갤러리 뷰로 표시되는가?
- [ ] Integration이 데이터베이스에 연결되어 있는가?
- [ ] 데이터베이스 ID에 하이픈이 포함되어 있는가?
- [ ] URL에서 `?` 앞의 ID를 정확히 복사했는가?

## 주의사항

- 페이지 ID와 데이터베이스 ID는 다릅니다
- 데이터베이스가 포함된 페이지의 ID가 아닌, 데이터베이스 자체의 ID를 사용해야 합니다
- 데이터베이스를 새로 만든 경우, URL을 다시 확인하세요
